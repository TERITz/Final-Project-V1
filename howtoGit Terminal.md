1. คำสั่งส่งงานขึ้น GitHub (Push)

เมื่อคุณเขียนโค้ดเสร็จแล้ว และต้องการส่งให้เพื่อนคนอื่นๆ เห็น ให้รันตามลำดับนี้ใน Terminal:
Bash

# 1. เช็คสถานะไฟล์ที่เราแก้
git status

# 2. เลือกไฟล์ทั้งหมดเตรียมส่ง (จุดคือทั้งหมด)
git add .

# 3. ใส่หมายเหตุว่าเราแก้อะไรไป (ภาษาไทยหรืออังกฤษก็ได้)
git commit -m "ใส่ข้อความอธิบายงานที่นี่"

# 4. ส่งขึ้น GitHub
git push origin main

(ถ้าเพื่อนใช้ branch ชื่อ master ให้เปลี่ยนคำว่า main เป็น master นะครับ)
-------------------------------------------------------------
2. คำสั่งดึงงานเพื่อนลงเครื่อง (Pull)

ทุกครั้งก่อนเริ่มทำงาน หรือเวลาเพื่อนบอกว่า "อัปงานแล้วนะ" ให้คุณรันคำสั่งนี้:
Bash

# ดึงโค้ดล่าสุดจาก GitHub มาทับไฟล์ในเครื่องเรา
git pull origin main
